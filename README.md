# UVCCameraDemo
UVC摄像头Demo（支持多个UVC摄像头）

## 概述
1.支持UVC格式多路摄像头。  
2.下载后请更换项目NDK为您本地r14版本。（NDK相关配置失败会导致黑屏无图）  
3.博客地址：https://blog.csdn.net/qq_24712507/article/details/92999760  

## 特点
1.运行后自动获取UVC设备信息，并展示在SurfaceView上。  
2.最多支持6个，再多的请自行修改UI及代码逻辑摄像头总数变量。
